## 🤔 这是什么？

一个 Claude Code Skill，安装后你的 AI 自动获得以下能力：

- 🎯 用李大霄的方式分析 A 股、港股、中概股
- 🗣️ 用地道的"大霄语录"说话（"同志们！""珍惜地平线下的宝贵时光！"）
- 📐 用**投资四面体模型**做系统性判断（宏观、市场、股票、你——四个维度一个不落）
- 🚫 自动发出黑五类警告（小盘、次新、垃圾、题材、伪成长——永远说不）
- 🍼 准确使用**底部命名体系**（钻石底→婴儿底→少年底→地平线）
- 💰 每次给出操作建议前，必问"用的是余钱吗？"

## 🎮 触发方式

**不需要手动触发。** 只要你聊到以下任何一个话题，大霄自动上线：

- 🏦 分析 A 股 / 港股 / 中概股
- 📊 问"这只股票能买吗？"
- 📉 讨论底部或顶部
- 💎 聊价值投资、蓝筹股、高股息
- 🏗️ 讨论银行股、央企、红利指数
- ⚠️ 加杠杆、仓位管理
- 🎢 市场恐慌或狂热

> 即便你完全没提"李大霄"三个字，只要聊股票，大霄就来。

## 📚 投资四面体（书中最核心的模型）

```
             宏观经济
              /  \
             /    \
            /  投  \
           /  资四  \
市场趋势 ────────────── 股票质量
          \  面体  /
           \     /
            \   /
             \ /
          投资者行为
```

四个维度相互制约。一个维度亮红灯，整体就危险。这就是为什么——你好不容易挑了一只十倍股，但用了杠杆、心态崩了，照样拿不住。

> **宏观决定有没有饭，市场决定什么时候吃，股票决定吃什么，行为决定你能不能消化。**
