# 选股框架：好股票 vs 黑五类

> "好公司 + 合理价格 = 好股票"
> —— 李大霄

---

## 一、好股票的五个标准

### 标准一：低估值

估值是李大霄选股的第一道筛子。他多次强调："再好的公司，买贵了也是差投资。"

| 指标 | 合理区间（李大霄标准）| 说明 |
|------|---------------------|------|
| PE | 行业历史下沿，最好低于 10 倍 | 10 倍 PE 是李大霄反复提及的"底部 PE" |
| PB | < 1.5，优选破净（< 1.0）| 破净意味着以低于净资产的价格买入 |
| 历史分位数 | 低于 10 年历史中位数 50% | 越接近历史极低估值越好 |

> 大霄说："2024 年 9 月 18 日上证 2689 点时，许多核心资产市净率仅 0.3-0.5 倍，便宜得离谱。"

### 标准二：高分红（真金白银）

李大霄对分红有着近乎执着的重视。在他眼中，不分红的公司——哪怕账面利润再漂亮——都是"铁公鸡"。

| 维度 | 标准 |
|------|------|
| 分红连续性 | 连续 5 年以上现金分红 |
| 股息率 | > 3% 合格，> 5% 优秀 |
| 分红来源 | 来自经营现金流，而非借款分红 |

> 大霄说："好股票是真金白银分红的。让所有股东赚钱的股票才是好股票。"

### 标准三：真盈利（非账面利润）

大霄非常关注利润的真实性：

| 检查项 | 标准 |
|--------|------|
| 扣非净利润 | 连续 3 年以上为正、稳定增长 |
| 经营现金流 | 覆盖净利润（现金流/净利润 > 1）|
| 应收账款 | 不异常膨胀 |
| 商誉 | 不超过净资产 30% |

> 大霄说："利润能真正变成现金的公司，才值得托付。"

### 标准四：好公司

什么是大霄眼中的"好公司"？不是增长速度最快的，而是最稳健、最让人安心的。

| 维度 | 标准 |
|------|------|
| 所有权性质 | 央企/国企优先（有国家信用背书）|
| 行业地位 | 行业龙头，市场份额稳定 |
| 商业模式 | 清晰可理解（大霄说"一页纸能讲清楚怎么赚钱"）|
| 管理层 | 诚信经营，不瞎折腾（不乱并购、不蹭概念）|

### 标准五：合理价格

好公司不等于随时可以买。大霄强调"合理价格"：

- 不在牛市狂热时追高
- 不在市场情绪亢奋时冲进去
- 在市场恐慌、估值处于历史低位时，分批建仓
- "底部是一个区域，不是一个点"——不用苛求买在最低点

## 二、黑五类详细定义

"黑五类"是李大霄独创的分类体系，用于标记五类他认为散户应该远离的股票。

### 第一类：小盘股

**定义**：市值过小（通常 < 50 亿），流动性差，容易被游资操纵。

**为什么远离**：
- 流动性差：大资金进出困难，散户容易被套
- 易被操纵：小市值股票不需要多少资金就能拉涨停或砸跌停
- 业绩不确定性大：小公司抗风险能力弱

### 第二类：次新股

**定义**：上市不满 3 年的股票。

**为什么远离**：
- 业绩未经时间验证——IPO 时的漂亮财报未必能持续
- 解禁压力——大量限售股解禁可能导致股价承压
- 估值偏高——A 股次新股上市初期往往高估
- "三年新，三年旧"——大霄认为起码要上市三年后才能看清一个公司的成色

### 第三类：垃圾股

**定义**：ST/*ST 股票，连续亏损，面临退市风险。

**为什么远离**：
- 基本面无价值——持续亏损的企业没有持有价值
- 退市风险——一旦退市，本金可能血本无归
- 炒作性质——垃圾股的上涨往往是游资拉高出货的陷阱
- "赌重组"是危险的赌博心态

### 第四类：题材股

**定义**：蹭概念热点、无实质业绩支撑的股票。

**为什么远离**：
- "风来了猪也能飞，风停了摔死的也是猪"
- 题材轮动快，散户跟不上节奏
- 等你能买进去的时候，可能已经是最后一棒
- 概念股的故事经不起时间考验

### 第五类：伪成长股

**定义**：用高 PE 包装成"成长股"，但增长不可持续。

**为什么远离**：
- 真成长 vs 伪成长：真成长有现金流支撑，伪成长只有故事
- 高 PE 意味着高期待，一旦"成长"不及预期，戴维斯双杀
- A 股历史上大量的"成长股"最后都原形毕露

## 三、选股决策树

```
第一步：黑五类筛查
    ├── 是否属于黑五类？ → 是 → 远离，不看第二眼
    └── 否 → 进入第二步

第二步：估值水位
    ├── PE 是否在历史低位？ → 否 → 等
    └── 是 → 进入第三步

第三步：分红质量
    ├── 连续 3 年以上分红？ → 否 → 减分，警惕
    └── 是 → 进入第四步

第四步：盈利真实性
    ├── 扣非净利润连年为正？ → 否 → 警惕
    ├── 经营现金流 > 净利润？ → 否 → 盈利质量存疑
    └── 是 → 进入第五步

第五步：行业与公司质地
    ├── 行业龙头或央企/国企？ → 否 → 给安全边际更大折扣
    └── 是 → 进入第六步

第六步：价格是否合理
    ├── 当前价格在估值低位？ → 否 → 等，或者以更严格标准审视
    └── 是 → 可以关注

最终判断 → 输出：可以买 / 等一等 / 坚决不碰
```

## 四、李大霄偏好行业

| 行业 | 为什么偏好 | 典型参考 |
|------|-----------|---------|
| **大型银行** | PB 长期破净、PE 4-7 倍、股息率 > 5%、有国家信用背书 | 工行、建行等大行 |
| **央企** | 经营稳健、不易爆雷、估值处于历史低位 | 央企指数成分股 |
| **公用事业** | 现金流稳定、需求刚性、高股息传统 | 电力、水务等 |
| **红利指数** | 天生具备"低估值 + 高分红"特征 | 中证红利等指数 |
| **保险** | 估值低、长期受益于利率上行 | 大型保险公司 |
