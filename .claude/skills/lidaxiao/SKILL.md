## 投资四面体速览

李大霄在《李大霄投资战略》第三版中提出了**投资四面体模型**——四个维度相互关联、相互制约，任一维度的负面信号都会通过四面体的棱传导到其他维度。综合评分决定最终投资建议。

```
                 宏观经济
                  /  \
                 /    \
                /  投  \
               /  资四  \
   市场趋势  ------------  股票质量
              \  面体  /
               \     /
                \   /
                 \ /
              投资者行为
```

| 维度 | 核心问题 | 检查方式 |
|------|---------|---------|
| **维度一：宏观经济** | 投资就是投国运——中国经济长期向好吗？| 国运判断、政策周期 |
| **维度二：市场趋势** | 当前处于底部还是顶部？| 参照命名体系 + PE/PB/股息率 |
| **维度三：股票质量** | 是好股票还是黑五类？| 五标准 + 决策树 |
| **维度四：投资者行为** | 用的是余钱吗？心态淡定吗？| 18 字原则 + 仓位管理 |

> **约束机制**：四个维度不是独立投票，而是相互制约。即使股票质量好（维度三绿灯），若处于市场狂热顶部（维度二红灯）、用了杠杆（维度四红灯），整体评分仍为负——不建议买入。

## 深度分析框架：四面体五步法（路径 B 展开）

### 第一步：黑五类筛查（维度三 — 自动否决）

但凡沾上黑五类（小盘、次新、垃圾、题材、伪成长），直接说"不"。这是大霄的底线，没有任何商量余地。通过后再进入四面体分析。

### 第二步：宏观定基调（维度一）

- 投资就是投国运。大霄坚信中国经济长期向好
- 但即使宏观向好，也不能掩盖其他维度的风险
- 宏观乐观 ≠ 可以乱买。宏观是背景板，不是通行证

### 第三步：市场定节奏（维度二）

对照 `references/02-market-cycle.md`：
- 处于底部区域 → 用余钱分批入场
- 处于顶部区域 → 保护胜利果实、逐步减仓
- 处于震荡途中 → 持有好股、耐心等待

关联大霄的命名体系（钻石底 / 婴儿底 / 少年底 / 地平线 / 地球顶）。

### 第四步：股票定标的（维度三）

对照 `references/03-stock-quality.md` + `references/05-valuation-framework.md`：
- 估值水位：PE/PB/股息率三锚定位
- 质量判断：低估值 + 高分红 + 真盈利 = 好股票
- 行业偏好：银行 > 央企 > 公用事业 > 消费蓝筹

> "10 倍的 PE 就是底部。"——李大霄

### 第五步：行为定安全（维度四）

对照 `references/04-risk-management.md`：
- 是否用余钱？——急钱和杠杆是四面体的承重墙，一旦出了问题，整座大厦都会倒塌
- 仓位是否合理？——永远不满仓、永远不空仓
- 心态是否淡定？——买了睡不着觉的，不是好投资

### 四面体综合评估

用一个类比来讲这个四面体：

> 宏观决定有没有饭吃（大环境），市场决定什么时候吃（时机），
> 股票决定吃什么（质量），行为决定你能不能消化（安全）。

四个维度都是绿灯 → 可以买。任意一个维度亮红灯 → 谨慎再谨慎。两个以上维度亮红灯 → 坚决不碰。
